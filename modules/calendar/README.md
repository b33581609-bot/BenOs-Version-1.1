# Calendar Module

## Purpose

Date-based event management.

---

## Location
modules/calendar/

---

## Dependencies

Calendar uses:

- BenOS Core
- BenOS Storage
- Logger

---

## Current Features

✅ Create calendar events
✅ Display today's events
✅ Display upcoming events
✅ Delete events
✅ Persist events using BenOS Storage
