# 🌲 BenOS

**Version:** 1.1 – Strong Roots

## Overview

BenOS is a personal productivity dashboard designed and built by Ben.

Its purpose is to provide one place to organize daily life, stay focused on long-term goals, and continue growing both personally and professionally.

The project is built from scratch using HTML, CSS and JavaScript while learning software development step by step.

---

## Current Features

### Dashboard
- Live Clock
- Live Date
- Dynamic Greeting

### Planner
- Today's Tasks
- Tomorrow's Tasks
- Start My Day
- Progress Tracking
- Auto Save
- Editable Tasks
- Delete Tasks

### Weather
- Live Weather
- Automatic Location Detection
- Barrie Fallback
- Current Conditions
- High / Low Temperatures

### Teachings
- Daily Seven Grandfather Teaching
