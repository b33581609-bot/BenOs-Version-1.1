# 🌲 BenOS Development Workflow

Every development session follows these steps.

---

## 1. Plan

- Review Roadmap.md
- Decide what feature will be built
- Break it into small tasks

---

## 2. Build

- Write the code
- Keep changes small
- Test after each step

---

## 3. Test

- Verify the feature works
- Check the browser console for errors
- Fix any bugs

---

## 4. Document

Update:

- Roadmap.md
- CHANGELOG.md
- README.md (if needed)

---

## 5. Version Control

git add .

git commit -m "Describe what changed"

git push

---

## 6. Finish

- Verify GitHub is updated
- Decide the next feature
- Record it in Roadmap.md

Session complete.
